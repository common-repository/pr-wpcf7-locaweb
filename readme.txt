Plugin de integração do Contact Form 7 com o sistema de Email Marketing da Locaweb.

== Description ==

Este é um plugin que auxilia na integração entre os formulários de contato gerados através
do Contact Form 7 e o sistema de envio de Email Marketing da Locaweb de maneira rápida, segura e confiável.

== Installation ==

1. Envie o arquivo `pr-wpcf7-locaweb.zip` para o diretório `/wp-content/plugins/`
1. Ative o plugin através do menu Plugins do WordPress
1. Crie as integrações através do sub-menu PR WPCF7 Locaweb dentro do menu do Contact Form 7
